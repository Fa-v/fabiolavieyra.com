30-Day JavaScript Challenge

Client: Personal Project

Made With: Vanilla JavaScript, HTML, CSS

Time in Development: 30 days, about an hour per project

GitHub repository

I completed the 30-day vanilla JS challenge. It was really fun working on it. It’s helped me to strengthen basic concepts about interacting with the DOM without libraries, just plain JavaScript and CSS.


I remeber when I started to learn to code last summer, one of the first things that blew my mind was the DOM concept. I didn’t really understand how it was possible to access an element you see on the screen and modify it. It was like magic for me, sounds crazy doesn’t it?


Just before doing this 30-day challenge I took a MEAN stack course (MonogoDB, Express, Angular, Node) which was really demanding for me. Yes, I know, I was trying to run before learning to walk. After going through the complexity of Angular 2, comming back to the basics with these exercises unraveled a lot of things in my head.


There is no better way to learn to code than through building things, so it was rewarding to get to build a small but challenging project every day.


I’ve personilized some exercises just for fun, but the core concepts are the same as presented by Wes Bos. Here are some of the projects I enjoyed:

 

Clock

A simple JavaScript clock, I was amazed at the simplicity of the code using the JavaScript Date instance.

 

See the Pen Clock by Fabiola Vieyra (@Fa-v) on CodePen.

 

Countdown Timer

Yes, another clock! This time more a countdown timer than a clock. You can set the timer with different intervals of time from 20 seconds to one hour. Again, thanks to the JavaScript Date instance and a bit more complicated logic we were able to build this countdown timer.

 

See the Pen Countdown clock by Fabiola Vieyra (@Fa-v) on CodePen.

 

Canvas

Canvas is an HTML element used to draw graphics on the fly using JavaScript.

 

See the Pen Canvas by Fabiola Vieyra (@Fa-v) on CodePen.

 

Unreal Webcam

This time we played again with canvas and the navigator object which contains information about the browser. In this case we use it to access the user's cammera.

 

See the Pen Unreal Webcam by Fabiola Vieyra (@Fa-v) on CodePen.
