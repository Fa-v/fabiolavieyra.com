Weather app

GitHub repository

The weather app is the second of the intermediate front-end development projects for Free Code Camp. I made this project on Codepen using JQuery in the summer 2016, which is broken now because browsers require HTTP secure connection to obtain user's geolocation - you can't serve pages with cross-origin resource sharing (CORS) conflicts. I decided to rebuild it instead as an Angular 2 app, so I could keep learning about Angular.


I decided to keep working with the OpenWeatherMap Api, but unfortunatly they don't provide secure http services for free accounts like mine. So, you can access the app after allowing geolocation on your browser and enabling the "not secure connection" option. This is, of course, something I need to work on:


To publish my app I found that the command 'ng github-pages:deploy' I used a couple of months ago has been removed from the CLI as of 1.0.0-beta.31. Instead, I used angular-cli-pages which is a pretty straightforward way to do this process.

Usé Harp para la estructura de este sitio y Bulma para los estilos
