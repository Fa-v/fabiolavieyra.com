Random Quote Machine

Client: Personal Project

Made With: JQuery, HTML, CSS

Time in Development: About three days

Go to my project in CodePen

The random quote machine is the first of the intermediate front-end development projects for Free Code Camp. The users stories to build this app are:

  • User Story: I can click a button to show a new random quote.
  • User Story: I can press a button to tweet out a quote.

Usé Harp para la estructura de este sitio y Bulma para los estilos
