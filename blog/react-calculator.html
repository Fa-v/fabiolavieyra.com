React Calculator

Client: Personal Project

Made With: React, Github Pages

Time in Development: About one week

Demo

GitHub repository

The Calculator is the first of the advance front-end development projects for Free Code Camp. I decided to use React to build this project.

 

I started to learn React while working on this project. I liked that getting started with it was easy because you work directly with JavaScript - React doesn't have many layers of abstraction. I used Create React App to bootstrap the app and published it on gh-pages.
