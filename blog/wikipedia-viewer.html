Wikipedia Viewer

Client: Personal Project

Made With: JQuery, HTML, CSS

Time in Development: About three days

Go to my project in CodePen

The Wikipedia viewer is the third of the intermediate front-end development projects for Free Code Camp. The user stories to build this app are:

  • User Story: I can search Wikipedia entries in a search box and see the resulting Wikipedia entries.
  • User Story: I can click a button to see a random Wikipedia entry.

Usé Harp para la estructura de este sitio y Bulma para los estilos
